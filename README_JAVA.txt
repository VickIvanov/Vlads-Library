КОСМИЧЕСКАЯ БИБЛИОТЕКА - JAVA ВЕРСИЯ
=====================================

ЧТО ЭТО:
- Веб-приложение на Java Spring Boot
- Работает в браузере
- Все функции сохранены

ТРЕБОВАНИЯ:
1. Java JDK 17 или выше
2. Maven 3.6 или выше

УСТАНОВКА:
1. Установите Java с oracle.com или adoptium.net
2. Установите Maven с maven.apache.org
3. Добавьте Maven в PATH

ЗАПУСК:
1. Дважды кликните: run-java.bat
2. Или в командной строке: mvn spring-boot:run
3. Откройте: http://localhost:8080

ФУНКЦИИ:
- Добавление книг с ID
- Удаление книг  
- Выбор фона
- Регистрация и вход
- Работа с .env файлами

ПРОБЛЕМЫ С КОДИРОВКОЙ:
Если видите странные символы в консоли - это нормально.
Это не влияет на работу программы.
Просто используйте английские команды или запускайте через .bat файл.
