# Инструкция по запуску сервера

## Если сервер не запускается автоматически:

1. Откройте терминал (PowerShell или CMD) в папке проекта:
   ```
   cd C:\Users\vladi\cosmic-library
   ```

2. Запустите команду:
   ```
   npm run dev
   ```

3. Дождитесь сообщения:
   ```
   ▲ Next.js 16.x.x
   - Local:        http://localhost:3000
   ```

4. Откройте браузер и перейдите на:
   ```
   http://localhost:3000
   ```

## Если порт 3000 занят:

Next.js автоматически попробует использовать следующий свободный порт (3001, 3002 и т.д.).
Смотрите в терминале, какой порт был выделен.

## Возможные проблемы:

1. **ERR_CONNECTION_REFUSED** - сервер не запущен
   - Проверьте, что команда `npm run dev` выполняется
   - Убедитесь, что нет ошибок в терминале

2. **Порт занят** - другой процесс использует порт 3000
   - Закройте другие приложения
   - Или используйте другой порт: `npm run dev -- -p 3001`

3. **Ошибки модулей** - зависимости не установлены
   - Выполните: `npm install`
