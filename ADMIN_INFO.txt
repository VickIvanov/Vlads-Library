ИНФОРМАЦИЯ ДЛЯ АДМИНИСТРАТОРА
=============================

ДОСТУП К АДМИН ПАНЕЛИ:
- URL: http://localhost:5000
- Логин по умолчанию: admin
- Пароль по умолчанию: admin123

ИЗМЕНИТЬ ПАРОЛЬ:
Создайте файл .env в корне проекта:
ADMIN_USERNAME=ваш_логин
ADMIN_PASSWORD=ваш_пароль

ПУБЛИЧНАЯ СТРАНИЦА:
- URL: http://localhost:5000/books
- Доступна всем пользователям
- Показывает все книги
- Есть поиск по названию, автору, жанру

ФУНКЦИИ:
- Только админ может добавлять/удалять книги
- Все пользователи могут просматривать каталог
- Книги сохраняются в папку "books"
- ID книги = название книги (без спецсимволов)
