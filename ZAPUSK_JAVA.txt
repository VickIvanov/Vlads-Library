ПРОСТАЯ ИНСТРУКЦИЯ ПО ЗАПУСКУ
=============================

ЭТИ СТРАННЫЕ СИМВОЛЫ - это просто проблема кодировки Windows.
Это НЕ влияет на работу программы!

КАК ЗАПУСТИТЬ:

1. Убедитесь что установлены:
   - Java JDK 17 или выше (проверьте: java -version)
   - Maven (проверьте: mvn -version)

2. Если Maven установлен:
   - Дважды кликните на: run-java.bat
   - Или откройте командную строку и выполните:
     mvn spring-boot:run

3. Откройте браузер:
   http://localhost:8080

ГОТОВО!

Если Maven не установлен:
- Скачайте с maven.apache.org
- Или используйте файл INSTALL_JAVA.txt для подробных инструкций
