ИНСТРУКЦИЯ ПО ЗАПУСКУ FLASK СЕРВЕРА
====================================

1. Установите Python (если еще не установлен)
   Скачайте с python.org

2. Установите Flask:
   Откройте командную строку (CMD) в папке проекта и выполните:
   pip install -r requirements.txt

3. Запустите сервер:
   
   Вариант А: Дважды кликните на файл run.bat
   
   Вариант Б: В командной строке выполните:
   python app.py

4. Откройте браузер и перейдите на:
   http://localhost:5000

5. Готово! Вы можете:
   - Добавлять книги
   - Выбирать фон
   - Управлять библиотекой

Для остановки сервера нажмите Ctrl+C в окне командной строки

НАСТРОЙКА .ENV ФАЙЛА
====================

Создайте файл .env в папке проекта со следующим содержимым:

LIBRARY_USERS=admin:admin123,user:user123
LOCAL_BACKGROUNDS=local1.svg,local2.svg,local3.svg

Для Vercel добавьте переменные окружения в настройках проекта.
