КАК ЗАПУСТИТЬ СЕРВЕР
====================

ПРОСТОЙ СПОСОБ:
1. Дважды кликните на файл: START.bat
2. Дождитесь сообщения "Сервер запущен!"
3. Откройте браузер: http://localhost:5000

ЧТО ОЗНАЧАЮТ СТРАННЫЕ СИМВОЛЫ:
- Это проблема кодировки Windows PowerShell
- Это НЕ ошибка программы
- Программа работает нормально
- Просто игнорируйте эти символы

ЕСЛИ СЕРВЕР НЕ ЗАПУСКАЕТСЯ:
1. Убедитесь что установлен Python
2. Откройте командную строку в папке проекта
3. Выполните: pip install Flask python-dotenv
4. Затем: python app.py

СЕРВЕР БУДЕТ РАБОТАТЬ НА:
http://localhost:5000
