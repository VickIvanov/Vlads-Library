# Фоны для библиотеки

## Для localhost
Поместите файлы фонов в эту папку. 
Поддерживаемые форматы: .jpg, .png, .webp

Примеры имен файлов:
- local1.jpg
- local2.jpg
- local3.jpg

## Для Vercel
На Vercel используйте переменную окружения `VERCEL_BACKGROUNDS` для указания списка фонов.
Например: `VERCEL_BACKGROUNDS=space1.jpg,space2.jpg,space3.jpg`

Файлы должны быть загружены в папку `public/backgrounds/` и доступны по URL `/backgrounds/filename.jpg`
