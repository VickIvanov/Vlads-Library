ПРОСТАЯ ИНСТРУКЦИЯ
==================

1. Дважды кликните на файл: run.bat

2. Дождитесь сообщения:
   "Сервер запущен!"
   "Откройте браузер: http://localhost:5000"

3. Откройте браузер и перейдите на:
   http://localhost:5000

ГОТОВО!

Если не работает:
- Убедитесь что установлен Python
- Откройте командную строку в папке проекта
- Выполните: pip install -r requirements.txt
- Затем: python app.py
